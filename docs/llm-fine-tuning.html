LLM fine-tuning

We must discover how to train a better LLM for Lettuce, using the power of a Lettuce federation!

Lettuce works well, but fine-tuning the LLM for Lettuce tasks will make it even better. We can't just use the input from a single user to fine-tune the model, or risk baking in the biases of one person. By using input from multiple users, we risk exposing data that needs to be kept secure.


Our solution is to train Lettuce in a federated setting, using BitFount.
